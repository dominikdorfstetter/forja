# My Drafts

The My Drafts page gives you a single view of all your work-in-progress content across every site you have access to. Instead of navigating into each site's blog or page listing, you can see all of your unpublished items in one place.

## Accessing My Drafts

Click **My Drafts** in the sidebar. The link is always visible regardless of which site is currently selected.

## What Is Shown

My Drafts lists every content item that meets all of the following criteria:

- **Status is Draft** -- only unpublished items appear here.
- **Created by you** -- only content authored by your account is included.
- **Across all sites** -- drafts are aggregated from every site you have access to, not just the currently selected site.

Both blog posts and pages are included.

## Draft List

Each entry in the list displays:

| Column | Description |
|--------|-------------|
| **Title** | The content title (or "Untitled" if no title has been set). |
| **Type** | Whether the item is a blog post or a page. |
| **Site** | The site the draft belongs to. |
| **Last Modified** | When the draft was last edited, shown as a relative timestamp. |
| **Edit** | A direct link to open the draft in the editor. |

## Editing a Draft

Click the **Edit** link or the draft title to open it in the corresponding editor (blog editor or page editor). After editing, you can either save your changes and keep the item as a draft, or publish it directly.

## Tips

:::tip
Use My Drafts as your daily starting point to pick up where you left off. It surfaces everything you still need to finish, regardless of which site it belongs to.
:::

- If you do not see an expected draft, make sure you are signed in with the account that created it -- My Drafts only shows your own items.
- Published or archived content does not appear here. To find those items, use the blog or page listing within the relevant site.
